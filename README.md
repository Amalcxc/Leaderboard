# Leaderboard

Leaderboard

# Built With

- HTML
- CSS
- JavaScript
- webpacker

## Setup

### Clone this repository

```bash
$ git clone https://github.com/Amalcxc/Leaderboard.git
$ cd Leaderboard
```

### Run project

```bash
$ npm install
$ npm run start # this will make webpack watching for your changes in code
```

### Open page in browser
```bash
$ npx webpack server
```
